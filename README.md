## TSVs
We have three TSV files that hold the data, [UMR aspect data](https://github.com/clairepost/UMR_Aspect_Data/blob/main/umr_aspect_data.tsv) holds the entire annotated dataset,[pear story](https://github.com/clairepost/UMR_Aspect_Data/blob/main/pear_story_practice.tsv) was practice for the annotators, and [difficult examples](https://github.com/clairepost/UMR_Aspect_Data/blob/main/difficult_examples.tsv) holds some of the most challenging predicates from the dataset for the annotators that were reviewed with experts. 

## PDF 
The [aspect annotition](https://github.com/clairepost/UMR_Aspect_Data/blob/main/Aspect%20Annotation.pdf) PDF was a learning tool provided by one of the annotators for the others to help assist in understanding and improve agreement in annotation.
